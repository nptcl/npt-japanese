# Constant-Variable LAMBDA-LIST-KEYWORDS


Constant Variable `LAMBDA-LIST-KEYWORDS`


## 定数値

リストであり、その要素は実装依存です。
しかし、少なくとも次のシンボルは含める必要があります。
`&allow-other-keys`, `&aux`, `&body`, `&environment`,
`&key`, `&optional`, `&rest`, `&whole`。


## 定義

実装で使われるすべてのラムダリストキーワードのリストです。
マクロ定義フォームでのみ使われるものも追加で含まれます。

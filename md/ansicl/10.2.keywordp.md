# Function **KEYWORDP**


Function `KEYWORDP`


## 構文

`keywordp` *object* => *generalized-boolean*


## 引数と戻り値

*object* - オブジェクト  
*generalized-boolean* - generalized-boolean


## 定義

もし*object*が型[`keyword`](10.2.keyword.html)なら*true*を、
それ以外なら*false*を返却します。


## 例文

```lisp
(keywordp 'elephant) =>  false
(keywordp 12) =>  false
(keywordp :test) =>  true
(keywordp ':test) =>  true
(keywordp nil) =>  false
(keywordp :nil) =>  true
(keywordp '(:test)) =>  false
(keywordp "hello") =>  false
(keywordp ":hello") =>  false
(keywordp '&optional) =>  false
```

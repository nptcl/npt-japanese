# System-Class **FILE-STREAM**


System Class `FILE-STREAM`

## 定義

型[`file-stream`](21.2.file-stream.html)のオブジェクトは、
ファイルから直接送受信するストリームです。
そのようなストリームは、
[`open`](21.2.open.html)と`with-open-file`によって明に作成されるか、
`load`のようなファイルを処理する関数で暗に作成されます。
